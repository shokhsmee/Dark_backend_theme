Dark Backend Theme

Dark navy & orange UI for Odoo 19 Community — no business logic, pure design

Overview

Dark Backend Theme is a fully standalone Odoo 19 backend theme. It replaces the default light interface with a professional dark navy + industrial orange design. Install it on any database, configure the branding from Settings, and get a consistent, polished UI — with zero impact on your business data.

No menus are added. No business models are touched. It is a pure UI layer that works alongside any existing or future addon.

Features

■ Navigation

  • Dark navy top navbar with orange accent border
  • Dynamic brand logo: custom icon + name from Settings
  • Back arrow to return to the home app grid
  • App section breadcrumbs preserved from Odoo core
  • Systray items (notifications, user menu) fully functional

■ Sidebar

  • Fixed 72px icon-only sidebar on the left
  • Orange left-border indicator for the active app
  • Smooth hover highlight with orange background tint
  • Shows PNG, SVG or Bootstrap Icons depending on each app
  • Hides automatically on mobile screens

■ Home App Grid

  • Full-screen 5-column responsive app grid
  • Dark navy background image for depth
  • Gradient card tiles with hover float animation
  • Orange icon accents inside each card
  • Collapses to 3 columns on tablet, 2 on mobile

■ Global Styling

  • League Spartan typography applied site-wide
  • Orange primary buttons with darker hover state
  • Orange accent on checkboxes, stat buttons, badges
  • Modal headers with orange bottom border
  • Clean search bar without visible borders
  • CSS variable --oil-primary controls all accent colors

How It Works

🔄

CSS Variable

A lightweight JS service fetches the configured color from ir.config_parameter before any component renders and sets --oil-primary on :root. All SCSS uses this variable, so the whole theme updates instantly.

⚙️

OWL Patch

The Odoo 19 NavBar component is extended via patch(). Brand name and icon are read from the theme config service and rendered reactively — no page reload required after navigating.

📷

XPath Extension

The sidebar and home app grid are injected using Odoo's standard t-inherit / xpath template extension pattern — fully compatible with future Odoo updates.

Installation

Via Odoo Apps Store

  1. Download the module ZIP from the Odoo Apps store
  2. Go to Settings → Apps → Update Apps List
  3. Search for Dark Backend Theme
  4. Click Install
  5. Reload the page to see the new look

Via Command Line

# Copy to your addons folder
cp -r dark_backend_theme /opt/odoo/extra-addons/

# Install on your database
python odoo-bin \
  -c odoo.conf \
  -d your_database \
  -i dark_backend_theme \
  --stop-after-init

# Restart Odoo
sudo systemctl restart odoo

Dependencies & Notes

  • web — Odoo core web module
  • base_setup — required for the Settings section
  • Bootstrap Icons 1.4.0 — loaded via CDN
  • League Spartan — loaded via Google Fonts CDN
Offline environments: Bootstrap Icons and League Spartan are fetched from external CDNs on each page load. If your server has no internet access, download these files, place them in static/src/, and update the @import URLs at the top of static/src/scss/backend_theme.scss.
